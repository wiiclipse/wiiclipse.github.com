WiiClipse

Eclipse CDT plugin for development of Nintendo Wii homebrew applications.

WiiClipse is a Eclipse CDT plugin that adds support to create and run Nintendo Wii homebrew applications using the devkitPro toolchain.

Requirements

  • Platform: Windows, Linux, Max
  • Eclipse IDE v3.6+
  • devkitPPC

Install

WiiClipse can be installed using the WiiClipse Eclipse Update Site

    http://wiiclipse.github.com/update-site/

(via Eclipse Help->Install New Software...)
